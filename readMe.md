# Personal Knowledge Management System

A web-based Personal Knowledge Management System developed using JSP, MySQL, HTML, CSS, and JavaScript to organize notes, study resources, and tasks efficiently.

## Features

* User authentication
* Dashboard management
* Subject and resource organization
* Search functionality

## Technologies Used

* JSP
* MySQL
* HTML/CSS
* JavaScript
